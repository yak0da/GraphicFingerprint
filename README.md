**Команда для запуска программы:**
    python3 fingerprint.py <отпечаток>
Пример:
    python3 fingerprint.py 3a1f8b4e2c9d5a7f1e3c8a6d4b2e9c7a


**Виды отпечатков**
1. Чистая 16-ричная последовательность
    Формат: Четная последовательность 16-ричных символов длиной от 24 до 64 символов (от 12 до 32 байт)
    Пример:
        0123456789AbcDeF0123456789

2. MD5-отпечаток
    Формат: 16-ричная последовательность с двоеточиями (от 12 до 32 байт)
    Пример:
        3a:1f:8b:4e:2c:9d:5a:7f:1e:3c:8a:6d:4b:2e:9c:7a
3. SHA256-отпечаток
    Формат: Последовательность байтов в base64.
            - Длина: 43 символа без padding или 44 символа с padding (=)
            - Поддерживаются символы: A-Z, a-z, 0-9, +, /
            - Поддерживается запись в виде SHA256:<отпечаток>
    Примеры:
        UxyIzKj93YXoldmfyP5K4/uKkHTu9UoG0nK9e93q57g
        SHA256:QmlgCt64NcKJ+aLTGGRM0hoMJcKQge/+hHv5CU/24Nk
